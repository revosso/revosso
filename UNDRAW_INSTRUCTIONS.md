# Instructions for Downloading unDraw Illustration

## Recommended unDraw Illustrations for Hero Section

Visit [https://undraw.co/illustrations](https://undraw.co/illustrations) and search for one of these:

1. **"Code Review"** or **"Code Contribution"** - Perfect for software development theme
2. **"Team Collaboration"** - Great for showing teamwork
3. **"Project Completed"** - Shows success and completion
4. **"Development"** - General development theme
5. **"Coding"** - Direct coding/development illustration

## Download Steps

1. Go to https://undraw.co/illustrations
2. Use the search bar to find one of the illustrations above
3. Click on the illustration you like
4. Click the "Download" button
5. Choose **PNG** format (or SVG if you prefer)
6. Set the color to match your brand (blue/purple gradient: use `#6366f1` or `#8b5cf6`)
7. Save the file as `hero-software-team.png` in `/public/images/` directory

## Alternative: Direct URLs

You can also try these direct URLs (replace `ILLUSTRATION_NAME` with the illustration name):
- https://undraw.co/illustrations/ILLUSTRATION_NAME

## Current Image

The code is currently set to use `/images/hero-software-team.png`. Make sure to replace the existing image file with your downloaded unDraw illustration.
